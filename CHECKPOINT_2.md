## 1. Текущий статус разработки ядра

В рамках текущего этапа полностью реализовано криптографическое ядро консольной утилиты. Написаны и протестированы следующие ключевые функции бизнес-логики:
* **Парсинг аргументов CLI:** Посредством `System.CommandLine` развернута обработка команд `encrypt` и `decrypt` со строгой валидацией входных путей.
* **Криптографический модуль (AES-256-GCM):** Настроено криптоядро с автоматической генерацией криптографически стойких векторов инициализации (Nonce) через `RandomNumberGenerator` и расчетом аутентификационных тегов (Tag) для защиты от модификации файлов.
* **Упаковка бинарных потоков:** Написана сериализация зашифрованных байт в кастомный защищенный формат файлов через `BinaryWriter`.

## 2. Интеграция стандартов обслуживания

### Изоляция секретов
Все потенциально уязвимые конфигурационные переменные вынесены из исходного кода в переменные среды окружения. Для развертывания подготовлен демонстрационный файл `.env.example`:

```text
# Шаблон конфигурации окружения для SecretGuard
SECRETGUARD_MASTER_KEY=ВашСуперНадежныйМастерПароль32Байта
```

### Логирование
Реализовано сквозное структурированное логирование с использованием библиотеки `Serilog`. Конфигурация ядра осуществляет разделение потоков данных:
* Системные логи уровня `INFO` и `DEBUG` выводятся в stdout контейнера и пишутся в файл `logs/info.log`.
* Ошибки и исключения уровня `ERROR` и `FATAL` изолируются в выделенный файл `logs/error.log`.

### Контейнеризация
Приложение полностью контейнеризировано. Настроен оптимизированный многоэтапный Dockerfile (Multi-stage build), разделяющий тяжелый SDK сборки и легковесный Runtime исполнения:

```dockerfile
FROM [mcr.microsoft.com/dotnet/sdk:8.0](https://mcr.microsoft.com/dotnet/sdk:8.0) AS build-env
WORKDIR /app

COPY src/SecretGuard/*.csproj ./src/SecretGuard/
RUN dotnet restore ./src/SecretGuard/SecretGuard.csproj

COPY src/SecretGuard/ ./src/SecretGuard/
RUN dotnet publish ./src/SecretGuard/SecretGuard.csproj -c Release -o out

FROM [mcr.microsoft.com/dotnet/runtime:8.0](https://mcr.microsoft.com/dotnet/runtime:8.0)
WORKDIR /app
COPY --from=build-env /app/out .
RUN mkdir logs

ENTRYPOINT ["dotnet", "SecretGuard.dll"]
```
